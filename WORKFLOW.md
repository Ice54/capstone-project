# Workflow Comparison: Vague vs. Precise Prompts

## Overview

I built the same todo list app twice using two different AI prompting strategies:

- **vague-prompt branch:** One sentence prompt, accepted output as-is

- **precise-prompt branch:** Detailed prompt with file structure, requirements, and verification

The goal was to see how prompt quality affects AI output quality and review effort.
